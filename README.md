# LLM-based Anomaly Detection Framework for NHANES Accelerometer Data

## Project Overview

This project implements a framework for detecting anomalies in NHANES accelerometer data (wearable device/fitness tracker) using Large Language Models (LLMs). The system is designed to identify anomalies in subjects' activity patterns on different workdays by leveraging a Teacher-Student knowledge distillation approach.

## Methodology

The framework utilizes a Teacher-Student knowledge distillation approach:

- **Teacher Model**: Employs FRIM (Functional Regression with Independent Margins), a statistical method. It processes the test data (`Data_test`) to generate anomaly detection results (`Test_anomaly`). These results serve as **soft labels** for the student model, encoding group-level anomaly patterns.
- **Student Model**: An LLM-based model that learns to detect anomalies by distilling knowledge from the teacher's soft labels. It aims to generalize these patterns to identify anomalies in unseen data.

## Dataset

The project uses accelerometer data from the National Health and Nutrition Examination Survey (NHANES) involving 2,080 subjects. The data is structured as follows:

| Dataset | Description | Purpose |
| :--- | :--- | :--- |
| **Data_train** | 4 workdays of "normal" activity data from 2,080 subjects. | Used to train the **Teacher Model** (FRIM) to establish a baseline of normal activity. |
| **Data_test** | Data from different workdays for the same 2,080 subjects. | The target data where anomalies need to be detected. |
| **Test_anomaly** | Anomaly detection results generated by the FRIM method on `Data_test`. | Serves as **soft labels** for training the **Student Model**, guiding it to learn group-level anomaly characteristics. |
